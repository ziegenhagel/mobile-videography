# Vorbereitung

## Mitbringen

- Smartphone oder Tablet mit FiLMiC Pro und Adobe Rush installiert
  - Bei Adobe Rush ist ein Login erforderlich
- Eine Sonnenbrille mit Gläsern, falls vorhanden.
